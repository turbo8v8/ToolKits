# Windows 系统下的工具

有时候，不得不额外准备Windows专用解决方案。

## 装机必备

- TortoiseSVN 版本管理
- Office~2007能满足就先用着，毕竟常用的也就那点功能。
- 浏览器（chrome，firefox）
- 输入法：搜狗吧
- WinMount 压缩包及虚拟硬盘读取，可惜停止更新了。
